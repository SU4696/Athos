Instrucciones de Juego
Utilizando las teclas A, W, S y D pordras mover el jugador.
"A" para moverte a la izquierda
"W" para moverte para arriba
"S" para moverte para abajo
"D" para moverte a la derecha
¿Que hacer para ganar?
Como jugador tendras que reponder las preguntas escogiendo la placa que tenga la respuesta correcta
En total responderas 20 preguntas
Al finalizar con las preguntas terminaras el juego y podras limpiar la piramide de su terrible maldicion
